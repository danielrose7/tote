## Notes on curation quality

- Titles should reflect real search intent, not marketing copy
- Each collection should have 10–20 items minimum for SEO density
- Descriptions should be specific ("for a small apartment") not generic ("for everyone")
- Clone-enabled so visitors can immediately start their own version
- See [CURATOR_WORKFLOW.md](./CURATOR_WORKFLOW.md) for how to run the curator
