# Use-Case Page Benchmark

Use this scorecard to benchmark current use-case pages, document why they scored that way, and track the next iteration target.

Scoring source: [USE_CASE_PAGE_RUBRIC.md](./USE_CASE_PAGE_RUBRIC.md)

## Rules

- Score fast; avoid over-calibrating
- Use integers from `0-5`
- Add one short note per category only when it changes what we should do next
- Prioritize the lowest score first
- Do not publish new pages unless Concrete Example >= 4 and Actionability >= 4
